# 前端开发角色库

## 角色定义

### UI/UX 设计师
- 职责：界面设计、交互设计、原型制作
- 输出：设计稿、交互说明、原型链接

### 前端开发
- 职责：组件实现、样式开发、响应式适配
- 输出：React/Vue 组件、CSS 样式、页面代码

### 前端架构师
- 职责：技术选型、组件架构、状态管理
- 输出：技术方案、组件结构图、架构文档

### QA
- 职责：UI 测试、兼容性测试、性能测试
- 输出：测试报告、兼容性数据、性能指标

### PM
- 职责：需求管理、进度跟踪

## 默认工作流
UI/UX 设计师设计 → 前端架构师规划 → 前端开发实现 → QA 测试

## 适用场景
- SPA 应用开发
- 组件库开发
- 响应式页面
- 前端重构
