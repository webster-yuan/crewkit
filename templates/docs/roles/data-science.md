# 数据科学角色库

## 角色定义

### 数据分析师
- 职责：数据探索、可视化、洞察提取
- 输出：分析报告、可视化图表、数据洞察

### 特征工程师
- 职责：特征设计、特征选择、特征管道
- 输出：特征列表、特征工程脚本、特征说明

### 模型工程师
- 职责：模型选型、训练调优、模型评估
- 输出：模型代码、训练脚本、评估报告

### MLOps
- 职责：模型部署、流水线构建、监控
- 输出：部署脚本、流水线配置、监控方案

### PM
- 职责：项目规划、实验跟踪、成果汇报

## 默认工作流
数据分析师探索 → 特征工程师设计 → 模型工程师训练 → MLOps 部署

## 适用场景
- 数据分析报告
- 机器学习建模
- 特征工程项目
- 模型部署流水线
