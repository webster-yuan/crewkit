# 后端开发角色库

## 角色定义

### 架构师
- 职责：系统架构设计、技术选型、接口规范
- 输出：架构图、技术方案、API 设计文档

### 后端开发
- 职责：接口实现、业务逻辑、数据库设计
- 输出：API 代码、数据库 schema、单元测试

### 安全工程师
- 职责：安全审计、漏洞分析、权限设计
- 输出：安全报告、权限方案、加固建议

### DevOps
- 职责：部署配置、CI/CD、监控告警
- 输出：Dockerfile、部署脚本、监控配置

### PM
- 职责：任务管理、进度跟踪

## 默认工作流
架构师设计 → 后端开发实现 → 安全工程师审计 → DevOps 部署

## 适用场景
- REST API 开发
- 微服务构建
- 数据库设计
- 后端系统重构
