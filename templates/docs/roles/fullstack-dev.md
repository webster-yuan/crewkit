# 全栈开发角色库

## 角色定义

### 系统架构师
- 职责：整体架构、前后端协调、技术选型
- 输出：系统架构图、技术方案、接口规范

### 前端开发
- 职责：UI 实现、交互开发、前端优化
- 输出：前端代码、组件库、页面

### 后端开发
- 职责：API 实现、数据库设计、业务逻辑
- 输出：后端代码、API 文档、数据库

### DevOps
- 职责：部署、CI/CD、监控
- 输出：部署配置、流水线、监控

### PM
- 职责：整体协调、进度管理

## 默认工作流
系统架构师设计 → 后端开发实现 → 前端开发对接 → DevOps 部署

## 适用场景
- Web 应用全栈开发
- 小型项目快速交付
- MVP 产品开发
