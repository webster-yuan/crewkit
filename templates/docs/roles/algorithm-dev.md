# 算法开发角色库

## 角色定义

### 算法研究员
- 职责：理论设计、数学建模、复杂度分析
- 技能：数学推导、算法选型、伪代码编写
- 输出：算法设计文档、公式推导、复杂度报告

### 数据工程师
- 职责：数据预处理、特征工程、数据管道
- 技能：数据清洗、特征提取、管道构建
- 输出：预处理脚本、特征说明、数据集

### 开发工程师
- 职责：算法实现、代码优化、工程化
- 技能：Python/C++实现、性能优化、API设计
- 输出：可运行代码、调用接口、使用示例

### 实验验证员
- 职责：设计实验、评估指标、对比测试
- 技能：实验设计、统计分析、可视化
- 输出：实验报告、性能数据、改进建议

### PM
- 职责：任务拆解、进度跟踪、成果汇总
- 输出：任务清单、进度报告、验收文档

## 默认工作流
算法研究员设计 → 开发工程师实现 → 实验验证员测试 → PM 汇总

## 适用场景
- 排序/搜索算法优化
- 推荐算法开发
- NLP/图像处理算法
- 数据结构设计
